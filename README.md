# holochain template project

This is a template for rapidly starting holochain projects.

## Steps

1. Clone this repo: `git clone https://github.com/lightningrodlabs/xyzzy`
2. Rename to you project: `mv xyzzy myproject`
3. `cd myproject`
4. Initialize:
